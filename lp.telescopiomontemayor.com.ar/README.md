# Basic instructions and configuration files for lp server

External repositories needed (install with add-apt-repository):

- ppa:deadsnakes/ppa
- ppa:certbot/certbot

Main packages:

    python3.7 python3.7-dev python3.7-venv python3.7-distutils virtualenvwrapper apache2-utils git-core curl gnupg2 ca-certificates lsb-release jq htop screen certbot python-certbot-nginx


Follow instructions from <https://nginx.org/en/linux_packages.html>, copy the configuration files and run certbot.
